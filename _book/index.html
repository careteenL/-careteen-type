type

English Document

JavaScript类型检查，弥补typeof不足

快速使用

npm i -D @careteen/type

import type from '@careteen/type'
type(null) // null
type(/a/) // regexp
type(new String('')) // string
type(new String(''), true) // String

更详细使用请查看API
